# Building a Next.js Blog

A guide to creating a modern blog with Next.js and TypeScript.

## Tech Stack

- **Next.js 16**: React framework
- **TypeScript**: Type safety
- **Tailwind CSS 4**: Styling
- **React Markdown**: Content rendering

## Project Structure

```bash
app/
├── components/     # Reusable UI components
├── lib/           # Utility functions
├── posts/         # Markdown blog posts
└── page.tsx       # Main page
```

## Features

- Directory-style navigation
- Markdown rendering
- Dark theme
- Terminal aesthetic
- Responsive design

## Getting Started

Install dependencies:

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to see your blog in action!
