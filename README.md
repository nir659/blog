# Minimal Blog

blog landing page built

## Quick Start
```bash
pnpm install
pnpm run dev
```

## Scripts
- `pnpm dev` — develop locally.
- `pnpm build` — create the production bundle.
- `pnpm start` — run the production build.
- `pnpm run lint` — lint with Next.js rules.

## Content
Markdown posts live under `app/posts/`; slugs mirror their relative paths and are served as plain text through `/api/posts/[...slug]`.
